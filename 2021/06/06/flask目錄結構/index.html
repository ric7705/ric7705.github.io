command

開啟debug模式:

  • it activates the debugger
  • it activates the automatic reloader
  • it enables the debug mode on the Flask application.

選擇執行目的:

執行

router

範例:

  • @app.route(‘/‘)
  • @app.route(‘/user/‘): url param用<>
    • 被裝飾的func要指名參數username
    • 或使用request, request.args.get(‘key’, ‘’)
  • @app.route(‘/post/int:post_id‘): 順便轉型
    • float, int, string
  • @app.route(‘/login’, methods=[‘GET’, ‘POST’]): 指名http method
    • func內搭配request.method使用

return

  • “string”: 回傳字串
  • redirect(url_for(‘login’)): redirect
    • 搭配url_for(): 轉一個url給你
  • render_template(‘hello.html’, name=name):使用template
  • abort(401): 終止接下來的動作, 順便拋http code
  • jsonify(): return json

cookie

  • request.cookies.get(‘username’): 取cookie
  • resp.set_cookie(‘username’, ‘the username’): 存cookie

系統設定

dev模式, 程式更改會自動重啟. /頁面會顯示錯訊

自訂config

有兩種方式匯入python檔案:

  • app.config.from_object()
    • 檔案內部放class, 可參考官網範例
  • app.config.from_pyfile()
    • 檔案內部放key=value, key須為大寫否則出錯
