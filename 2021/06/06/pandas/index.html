### 常用method

假設列為日期, 欄為A, B, C,...

df.head(), df.tail(3) => 取前後

df.index, df.columns, df.values => 取列label, 欄label, 表格值

df.sort_index(axis=1, ascending=False) => 根據列排序

df.sort_values(by='B') => 根據欄的某column排序

df.loc['20130102':'20130104',['A','B']] => 選定列範圍, 選定欄(列全選使用:即可)

df.iloc[3:5,0:2] => 類似loc, 但變成用index

df[df.A > 0] => A欄>0當filter

df.mean() => 各欄求平均

df.apply(lambda x: x.max() - x.min()) => 這個好像很多用法
